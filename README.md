# Brainsort

A brain-dump app for people with ADHD (and everyone else who has too many thoughts).

## What it does

- You type whatever is in your head — tasks, feelings, random thoughts
- It organises your dump into a clean to-do list
- For reflections, it asks a couple of gentle follow-up questions and turns it into a short journal entry
- Everything is saved so you can look back on it

## How it works

- Chat-style interface — just type like you're talking to someone
- Powered by Claude or ChatGPT (users bring their own API key)
- Accounts so your data is saved between sessions

## Tech stack

- **Frontend:** React
- **Backend:** Node.js + Express
- **Database:** Supabase
- **AI:** Anthropic Claude API / OpenAI API
